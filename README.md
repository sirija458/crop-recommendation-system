# 🌱 ML Crop Recommendation System

## Overview

A machine learning web application that recommends the most suitable crop based on soil nutrient values (N, P, K) and rainfall using the Random Forest algorithm. The application is built with Streamlit and provides an interactive interface for real-time predictions.

## Features

- Crop recommendation using Machine Learning
- Random Forest classification model
- Interactive Streamlit interface
- Model accuracy evaluation
- Crop distribution visualization
- User-friendly input form

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- Scikit-learn

## Project Structure

crop-recommendation-system/

## Installation

git clone https://github.com/sirija458/crop-recommendation-system.git

cd crop-recommendation-system

pip install -r requirements.txt

streamlit run app.py
